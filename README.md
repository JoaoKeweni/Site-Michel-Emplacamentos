# Michel Emplacamentos

-Acesse o site: https://michelemplacamentos.com.br
🟢Status: Em produção, no ar e totalmente funcional.

Site institucional desenvolvido em HTML, CSS Vanilla e JS para a empresa Michel Emplacamentos, com foco em soluções completas de despachante, emplacamento, ANTT, AET e cursos profissionalizantes.

> **Nota:** Este projeto foi desenvolvido e aperfeiçoado com o auxílio de ferramentas avançadas de Inteligência Artificial para alavancar a produtividade, iterando conforme fundamentos do modelo ágil e de *pair programming*.

## Contexto do Projeto

- **Público-alvo:** Despachantes procurando parcerias (venda de ANTT e cursos preparatórios) e clientes gerais com necessidades de emplacamento e documentação veicular.
- **Tecnologias Utilizadas:** HTML5 Semântico, CSS3 (variáveis, flexbox/grid, animações) e JavaScript Vanilla.
- **Identidade Visual:** Foco em confiança e agilidade, com layout responsivo (*Mobile-First*).
  - Azul Principal (`#1e3c8a`) e Azul Escuro (`#142a5e`) 
  - Amarelo Destaque (`#fefa0a`) e Branco (`#ffffff`)
- **Estilo:** Transições suaves, componentes em glassmorphism, micro-animações no *scroll* e botões unificados com ícones vetoriais padronizados e monocromáticos (SVGs integrados inteligentemente).

## Estrutura e Funcionalidades

1. **Home / Hero:** Banner com chamada focada ("Seu veículo regularizado com agilidade e segurança").
2. **Sobre Nós:** Exibição institucional com um *Slider Automático* moderno apresentando as instalações e a equipe.
3. **Serviços ANTT e Gerais:** Resumo em grade (*grid*) com hover responsivo nos SVGs monocromáticos de serviços de TAC, ETC, transferências, etc.
4. **Comunicação por Email Nativa:** Integração do formulário via API do **FormSubmit.co** com interações assíncronas (AJAX), preparando a base para uploads dinâmicos de documentações de clientes diretamente para a caixa de e-mail institucional.
5. **Navegação (Single Page):** Menu flutuante adaptável ao *scroll* vertical com indicadores ativos da seção corrente.
